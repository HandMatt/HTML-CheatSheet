Introduction to SQL

SQL stands for Structured Query Language. It is used to communicate with a database.

SELECT

SELECT is the clause used every time you want to query information from a database.

AS

AS renames a column or table.

DISTINCT

DISTINCT returns unique values.

WHERE

WHERE is a popular command that lets you filter the results of the query based on conditions that you specify.

LIKE and BETWEEN

LIKE and BETWEEN are special operators.

AND and OR

AND and OR combines multiple conditions.

ORDER BY

ORDER BY sorts the results.

LIMIT

LIMIT specifies the maximum number of rows that the query will return.

CASE

CASE creates different outputs.

Example:

SELECT name,
 CASE
  WHEN imdb_rating > 8 THEN 'Fantastic'
  WHEN imdb_rating > 6 THEN 'Poorly Received'
  ELSE 'Avoid at All Costs'
 END AS 'Review'
FROM movies;

The above code would produce query results from a movies database. The results will have two columnns, name and Review. Conditional statements are applied to the imdb_rating values. The Review column will have one of three string values.

SQL Database Manipulation

SQL is a programming language that can be used to manage data stored in a relational database. It is designed to manipulate and manage data.

  • A relational database is a database that organizes information into one or more tables.
  • A table is a collection of data organizes into rows and columns.
  • Constraints add information about how a column can be used.

CREATE TABLE

CREATE TABLE creates a new table.

INSERT INTO

INSERT INTO adds a new row to a table.

SELECT

SELECT queries data from a table.

ALTER TABLE

ALTER TABLE changes an existing table.

UPDATE

UPDATE edits a row in a table.

DELETE FROM

DELETE FROM deletes rows from a table.

Aggregate Functions

Aggregate functions are used to perform calculations on data.

COUNT()

COUNT(): count the number of rows.

SUM()

SUM(): the sum of the values in a column

MAX() / MIN()

MAX()/MIN(): the largest/smallest value.

AVG()

AVG(): the average of the values in a column.

ROUND()

ROUND(): round the values in the column.

Aggregate functions combine multiple row together to form a single value of more meaningful information.

GROUP BY

GROUP BY is a clause used with aggregate functions to combine date from one or more columns.

HAVING

HAVING limit the results of a query based on an aggregate property.

Multiple Tables

The below clauses allow for querying information from multiple tables

JOIN

JOIN will combine rows from different tables if the condition is true.

LEFT JOIN

LEFT JOIN will return every row in the LEFT table, and if the join condition is not met, NULL values are used to fill in the columns from the RIGHT table.

Primary Key

Primary key is a column that serves a unique identifier for th rows in the table.

Foreign Key

Foreign key is a column that contains the primary key to another table.

CROSS JOIN

CROSS JOIN lets us combine all rows of one table with all rows of another table.

UNION

UNION stacks one dataset on top of another.

WITH

WITH allows use to define one or more temporary tables that can be used in the final query.
